# 📚 Shopiko - Book Store App

## 🚀 About the Project

Shopiko is a full-stack bookstore web application where users can browse books, add them to cart, and place orders.

---

## 🛠️ Tech Stack

### Frontend:

* Next.js
* React
* CSS / Tailwind

### Backend:

* Node.js
* Express.js
* MongoDB

---

## ✨ Features

* User authentication (signup/login)
* Browse books
* Add to cart
* Favorite books
* Place orders

---

## 📂 Project Structure

```
Shopiko/
│── backend/
│── frontend/my-app/
```

---

## ⚙️ Installation

### 1. Clone the repo

```
git clone https://github.com/your-username/shopiko.git
cd shopiko
```

### 2. Install backend

```
cd backend
npm install
npm start
```

### 3. Install frontend

```
cd frontend/my-app
npm install
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in backend and add:

```
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```
